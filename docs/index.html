ClipVault

A privacy-first clipboard history manager for Windows.

ClipVault keeps a searchable history of what you copy, encrypts it at rest, and leaves no persistent system footprint.

Features

  • Encrypted at rest — clipboard content is encrypted per-field with ChaCha20-Poly1305. The master key is sealed with Windows DPAPI and can require a passphrase (Argon2id) or Windows Hello.
  • Volatile mode — optionally keep the entire history in RAM only; nothing touches the disk.
  • Privacy gate — API keys, JWTs, private keys, card numbers, and passwords are rejected or masked before they are stored. The OS "exclude from clipboard history" signal is honored, and you can exclude specific apps.
  • Lightweight & install-free — an unpackaged, self-contained .exe; tray-resident; summon with a global hotkey. OS registrations are runtime-only and released on exit.

Security

The history is encrypted per-field with ChaCha20-Poly1305 (AEAD); the master key is sealed with DPAPI and can require a passphrase or Windows Hello. Deleting the key file (panic wipe) renders all ciphertext unrecoverable. Security model →
