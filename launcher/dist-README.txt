ClipVault
=========

To start ClipVault, double-click  ClipVault.exe  in this folder.

Everything else lives in the  app\  subfolder. You do not need to open or
install anything there — ClipVault.exe launches it for you. Keep this folder
intact (ClipVault.exe and app\ together); you can move the whole folder
anywhere, including a USB drive.

Press Win+V to summon ClipVault once it is running. It lives in the system
tray (no window appears at startup).
